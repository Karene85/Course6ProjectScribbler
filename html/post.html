Everything you should know about ‘module’ & ‘require’ in Node.js

Srishti Gupta

Node.js treats each JavaScript file as a separate module. For instance, if you have a file containing some code and this file is named xyz.js, then this file is treated as a module in Node, and you can say that you’ve created a module named xyz. Let's take an example to understand this better. You have a file named circle.js which consists of the logic for calculating the area & circumference of a circle of a given radius. You can call circle.js file a module named circle.You might be wondering why there is a need to have multiple modules?
